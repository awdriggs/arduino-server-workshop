# Arduino and Web Servers

## Prerequisites

### Hardware
- Arduino Nano IOT
- A few 10k Reistors
- a few LEDs
- a phototransistor or other analog sensors
- a few push buttons
- Wifi connections

### Software
- Arduino IDE
- VScode (or another code editor)
- Node.js installed
- Install [ngrok](https://www.npmjs.com/package/ngrok) (`npm install ngrok`)
